## 707. 复制物件

### 707.1 复制的定义

**707.1.** 一些效应会复制物件。复制的物件的各特征，除了原物件所具有的少部分特征之外，与复制效应所定义的完全相同。

### 707.2 可复制特征值

**707.2.** 可复制特征值包括物件的以下特征值：名称、法术力费用、颜色、颜色标志、类别、副类别、超类别、规则叙述、异能、力量、防御力、忠诚度、法术力值、攻防比、以及任何由复制效应或结聚所定义的特征。可复制特征值不包括以下特征：物件是否是衍生物；物件是否是牌面朝下、被转化、被翻面、或是已转换；物件是否有面朝下的咒语或牌面朝下的永久物叠放在其上；以及指示物、贴附的灵气或武具、或任何非复制效应或非结聚效应对该物件所做出的其他变更。

### 707.3-707.5 复制永久物

**707.3.** 一些效应将永久物复制为衍生物，或将永久物复制为复制品。这些效应具有「[物件]的复制品」的字样。此类效应可以进入战场，或可以生成衍生物。

**707.4.** 一些效应将永久物复制为其操控者手牌中的牌。这只会复制该物件的可复制特征值。它不会复制其操控者，且（如果是复制品或衍生物）不是复制品或衍生物。

**707.5.** 一些效应将永久物复制为其拥有者牌库中的牌。它会成为该牌库中的一张新牌，且会成为其拥有者主牌表中的一张新牌。

### 707.9 替代性效应与复制

**707.9.** 被复制物件的替代性效应不会被复制。

### 707.10-707.12 复制咒语

**707.10.** 一些效应将咒语复制为复制品。此类效应在堆叠上创造该咒语的复制品。该复制品是咒语，即使在它结算之前原咒语的复制品就已经离开了堆叠。

**707.11.** 如果原咒语具有X费用，则复制咒语的X值等同于原咒语的X值。

**707.12.** 如果原咒语拥有模式选择，则复制咒语具有与原咒语相同的模式。

## 复制与层系统

复制效应在**层1**生效（见规则613.2a）。这意味着：

1. 层1：复制效应决定物件的特征
2. 层2-7：其他持续性效应再应用于已复制的特征

### 案例：Mocking Bird 的"作为...复制品进场"

Mocking Bird：「你可以让此生物作为战场上任何法术力值小于或等于施放此生物时花费的法术力数量的生物之复制品进场...」

**关键点**：这个"作为复制品进场"是**替代性效应**（规则614.1c），在层1生效。

**问题**：如果通过召令和弦（搜寻放进战场，非施放）让 Mocking Bird 进场：
- Mocking Bird 从未被施放
- "施放时花费的法术力数量" = 0
- 只能复制法术力值≤0的生物

## 常见误区

| 问题 | 答案 |
|------|------|
| 副本有指示物吗？ | 没有，指示物不是可复制特征 |
| 副本横置吗？ | 进入战场时是重置的 |
| 复制咒语需要支付费用吗？ | 不需要 |
| 复制的异能可以启动吗？ | 可以，副本具有所有异能 |
