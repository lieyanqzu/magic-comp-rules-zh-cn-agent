## 处罚等级

IPG 定义了四种处罚等级：

1. **警告 (Warning)** - 轻微违规
2. **一盘负 (Game Loss)** - 严重违规
3. **一局负 (Match Loss)** - 非常严重违规
4. **取消资格 (Disqualification)** - 恶意违规或作弊

## 常见违规类型

### 游戏行动失误 (Game Play Errors)
- 遗漏触发
- 额外看牌
- 非公开牌张失误
- 再调度失误
- 违反游戏规则
- 未维护游戏状态

### 比赛失误 (Tournament Errors)
- 迟到
- 外来协助
- 游戏进行过慢
- 套牌登记表问题
- 套牌问题

### 举止失误 (Unsporting Conduct)
- 轻微
- 严重
- 作弊
